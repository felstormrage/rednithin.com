# First Post

This site is generated with **Blogatto** and uses **Lustre** for the static pages.

## What to edit

- Update `src/me/site.gleam` to change your routes and layout.
- Add more posts under `blog/`.
- Adjust `static/style.css` to change the presentation.

## Build

Run `gleam run` to generate the site into `./dist`.
